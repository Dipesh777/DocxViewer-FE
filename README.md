## Installation

### Frontend Setup

1. **Clone the repository**
   ```bash
   git clone <FRONTEND_REPO_URL>

2. **Change directory**
   ```bash
   cd DocxViewer-FE

3. **Install dependencies**
   ```bash
   npm install

4. **Start the development server**
   ```bash
   npm run dev

5. **Open the app**
   ```bash
   Visit http://localhost:5173/ in your browser.

Note: The backend is already deployed on Railway. You do not need to run it locally. Simply use the frontend to open and view documents.

1. Technologies Used

2. Frontend: React, Vite

3. HTTP Requests: Axios

4. Backend: Deployed on Railway

5. Others: JWT for authentication (if applicable)
